# Step-by-Step Usage Guide

Follow the instructions below to configure and use the extension correctly.

### 1. Bypass the KeySystem URL
Before using the extension, you must first bypass the KeySystem URL using any online bypass service or a bypass tool of your choice. Once the bypass is complete, copy the generated URL, as you will need it in the next step.

### 2. Open the Original KeySystem Page
Navigate to the original KeySystem page in your browser. With the page open, launch the extension from your browser's toolbar.

### 3. Paste the Bypassed URL
Inside the extension, locate the text input field and paste the bypassed URL you copied earlier. Make sure the entire URL has been pasted correctly before continuing.

### 4. Select the Appropriate KeySystem
Below the input field, you will find a list of supported KeySystem providers. Select the option that matches the website or service you are currently using.

If your provider is not listed, simply select the **Custom** option instead.

### 5. Configure the Available Options
Further down, you will find two optional checkboxes. You may configure them according to your preference.

However, it is **strongly recommended** to leave both options enabled, as they are intended to reduce the likelihood of detection and improve compatibility.

### 6. Redirect
Once everything has been configured correctly, click the **Redirect** button at the bottom of the extension.

The extension will automatically process the information and redirect you to the appropriate page.

## Notes

- Not every website or KeySystem is vulnerable to this method.
- Some websites include additional security checks and may still detect or block the bypass.
- Compatibility depends on how the target website implements its KeySystem and anti-bypass protections.
